CursorPointer

Demo page

The CursorPointer JS module requires a browser that supports window.getComputedStyle. The browser should also support the Element method .getBoundingClientRect().

There are no dependencies on other scripts. If your browser supports those methods, then you should notice your cursor pointing to the button that wants your attention.

When that button has been clicked, the other button will ask for your attention, and the pointer will start pointing there.

To use the module, put a reference to cursor-pointer.js in your page. Then in your own script, use CursorPointer.pointAt(), sending the element you want the cursor to point at.
